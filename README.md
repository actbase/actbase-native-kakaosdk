- 카카오 사용자관리
- 카카오링크
- 카카오톡
- 카카오스토리
- 카카오톡 채널


## Getting started

`$ npm install actbase-native-kakaosdk --save`

### Mostly automatic installation (RN <= 0.59)

`$ react-native link actbase-native-kakaosdk`
